# ATS Keywords for SRE & DevOps

## Infrastructure as Code (IaC)
- Terraform, CloudFormation, Pulumi, Ansible, Chef, Puppet, SaltStack.

## Cloud Platforms
- AWS (EC2, S3, RDS, Lambda, EKS, IAM), Azure (AKS, Functions), Google Cloud (GKE, BigQuery).

## Containerization & Orchestration
- Kubernetes, Docker, Helm, Istio, Linkerd, OpenShift.

## CI/CD & DevSecOps
- GitHub Actions, GitLab CI, Jenkins, Spinnaker, ArgoCD, FluxCD, SonarQube, Snyk.

## Observability & Monitoring
- Prometheus, Grafana, ELK Stack, Splunk, Datadog, New Relic, OpenTelemetry, Jaeger.

## Reliability & Incident Management
- SLO, SLI, SLA, Error Budgets, Incident Response, Post-mortems, On-call, Root Cause Analysis (RCA).

## Automation & Scripting
- Python, Go, Bash, Java, JavaScript, AI-powered automation, Prober.

## Databases
- PostgreSQL, MySQL, MongoDB, Redis, Cassandra, SQL performance tuning.

## Networking & Security
- TCP/IP, VPC, DNS, SSL/TLS, OAuth, ITIL, SOC2 compliance.
